Die zwei Dateien in diesem Ordner sind zum einen in Python 2.x (tweet_catcher) und zum anderen in Python 3.x (hashtag_catcher) geschrieben. 
Da wir Anaconda 3.x heruntergeladen haben bedeutet dies fuer uns, dass wir um den tweet_catcher zu nutzen zuerst in unserem Anaconda Navigator eine virtuelle Umgebung (Environment) in Python 2.x starten muessen.

In dieser virtuellen Umgebung muessen wir dann nochmal ueber das Terminal/Command Prompt das Package Tweepy installieren.
Die Installation erfolgt ueber den folgenden Befehl: pip install Tweepy

Der andere Skript (hashtag_catcher) widerrum laeuft in unserer normalen Conda Prompt/ Terminal. Allerdings muss hier noch das Package pandas mit "pip install pandas" oder ueber die GUI vom Conda Navigator erstellt werden.


Siehe dazu auch Vorlesungsslides. 

Nutzen Sie die Twython Dokumentation unter https://twython.readthedocs.io/en/latest/ um den Code zu verstehen,
bzw. ihn auf Ihre Beduerfnisse anzupassen
